# Анализ опроса поведения пользователей мобильного приложения

Данные получены из публичного датасета с Kaggle “Mobile App User Behavior Dataset”. В нем хранятся данные опроса пользователей мобильного приложения.

**Цель проекта** провести EDA (устройство и платформа, демографический анализ, пользовательское поведение) и построить воронки поведения пользователей, чтобы узнать на каком этапе происходит самый большой отток клиентов, какие есть отличия поведения пользователей разных платформ и что нужно улучшать.
